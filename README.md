# Projek Tugas Akhir
Sistem Pemesanan Kamar Hotel yang dibuat dengan intellij untuk membuat tampilan dan sistem menginputkan data-data untuk keperluan pemesanan kamar hotel. Dari hasil menginputkan data lalu akan di simpan ke dalam database mysql

# 1. Tampilan Utama
Menampilakan Form Utama (halaman utama), menampilakan menu pilihan untuk sistem pemesanan kamar hotel ada beberapa pilihan. Saat mengklik salah satu menu akan menampilkan form lainya.
![image alt](https://github.com/yahyaef/Gui_Pemesanan/blob/1bd37fdfb0bf2434790859dc5e726b053a94fda7/menu.png?raw=true)

# 2. Tampilan Form Penginputan Biodata
Menampilkan Form Pemesanan kamar hotel dengan mengiputkan data identitas lalu disimpan di database dan ditampilkan di text area fungsinya untuk mengetahui data yang diinputkan sudah benar apa belum. Dan form ini juga untuk mengedit data identitas, menghapus data, menciri data.
![image alt](https://github.com/yahyaef/Gui_Pemesanan/blob/10671ad031d38f75c15fb40169c00dd662faf49a/biodata.png?raw=true)

# 3. Tampilan Form Pemilihan Kamar Hotel
Menampilkan Form Kamar digunakan untuk menginputkan data kamar yang akan mau di pesan. dibagian pemilihan tipe kamar saya buat program jika memilih salah satu tipe kamar akan langsung di tentukan harga dan akan tampil di jtextfield harga. Lalu menginputkan lama menginap, lalu klik hitung harga per malah dan lama menginap dikalikan, lalu akan tampil di jtextfield total. Selanjunya menginputkan tanggal masuk menggunakan jcalender. Form tamu menyimpan data dan menampilkan data di textarea, mengedit data, mencari data, menghapus data. Dan menampilkan data inputan mengunakan textarea.
![image alt](https://github.com/yahyaef/Gui_Pemesanan/blob/10671ad031d38f75c15fb40169c00dd662faf49a/kamar.png?raw=true)

# 4. Tampilan Form Transaksi
Form Transaksi ini digunakan menginputkan data tamu (identitas) dan data kamar, dengan cara memilih nik salah satu dan akan tampil di jlabel, dan data kamar caranya seperti tamu. Lalu akan mengiputkan bayar tidak bolek kurang dari total harga kamar.
![image alt](https://github.com/yahyaef/Gui_Pemesanan/blob/10671ad031d38f75c15fb40169c00dd662faf49a/transaksi.png?raw=true)

# 5. Tampilan Ditail Dan History Pemesanan Kamar Hotel
Form Detail History ini menampilkan data identitas dan data kamar yang telah di pesan
